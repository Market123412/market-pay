## 🎯 Pixel e Conversões

### Eventos para Rastrear
1. **ViewContent**: Visualização do produto
2. **AddToCart**: Clique no link afiliado
3. **Purchase**: Conversão (via Hotmart pixel)

### Configuração Pixel
```javascript
// Adicionar ao Next.js (se ainda não tiver)
import ReactPixel from 'react-facebook-pixel';

const pixelId = 'SEU_PIXEL_ID';
ReactPixel.init(pixelId);

// Eventos
ReactPixel.track('ViewContent', {
  content_name: 'Viver de Cílios Plano Master',
  content_category: 'beleza',
  value: 197.00,
  currency: 'BRL'
});
```

## 📊 Otimização Diária

### Métricas para Monitorar
- **CTR**: >2% (bom), <1% (pausar)
- **CPC**: R$0,50-R$1,50 (meta)
- **Conversão**: >1% (ótimo)
- **ROAS**: >2 (escalar)

### Ações Automáticas
- **CTR < 1%**: Pausar anúncio após 2 dias
- **ROAS > 3**: Aumentar orçamento em 20%
- **Conversão = 0**: Testar novo criativo

## 🚨 Dicas de Segurança

### Meta Ads Policy
- Evite "antes/depois" exagerado
- Não prometa resultados garantidos
- Use "pode ajudar" em vez de "vai ganhar"
- Declare conteúdo patrocinado

### Hotmart Compliance
- Links diretos (sem encurtadores)
- Não use "grátis" se for pago
- Respeite período de garantia
- Declare afiliação claramente
